# 基于SpringBoot+Vue的在线问卷调查系统

![Spring Boot](https://img.shields.io/badge/-Spring%20Boot-6DB33F?style=flat-square&logo=spring-boot&logoColor=white) ![Vue3](https://img.shields.io/badge/-Vue3-grey?style=flat-square) ![MySQL](https://img.shields.io/badge/-MySQL-4479A1?style=flat-square&logo=mysql&logoColor=white) ![MyBatis](https://img.shields.io/badge/-MyBatis-000000?style=flat-square&logo=mybatis&logoColor=white) 

## 项目介绍

InsightForm 是一款面向个人用户和管理员的在线问卷调查平台，支持多种题型设计、问卷发布审核、数据收集和统计分析等核心能力。

## 主要功能模块

- **用户认证模块**：支持用户注册、登录、密码修改，基于 Token + Session 双重认证
- **问卷管理模块**：问卷的创建、编辑、复制、删除，支持单选题、多选题、填空题三种题型
- **审核管理模块**：管理员对用户提交的问卷进行审核（通过/驳回），维护问卷内容质量
- **数据收集模块**：已审核通过的问卷生成分享链接，支持匿名填写和提交
- **统计分析模块**：按题目维度展示回答分布，柱状图+环形图可视化呈现，支持数据导出
- **后台管理模块**：管理员可管理用户角色、查看所有问卷状态及回答数据

## 技术架构

- **后端**：Spring Boot 2.6.13 + MyBatis 2.2.2 + Lombok + FastJSON
- **前端**：Vue 3.5 + Vue Router 4 + Axios + Vite 8
- **数据库**：MySQL 8.0
